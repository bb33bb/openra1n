# openra1n (UNOFFICIAL)
UNOFFICIAL palera1n booter for windows / s8003

## Dumped payload from palera1n v2.0.0 beta 7 
- pongoOS_shellcode(already compressed).bin 
- checkra1n-kpf-ploosh.bin
- ramdisk.bin
- overlay.bin

## Usage
0. Switch apple mobile device driver with libusbK
1. openra1n.exe
2. python3 boot.py
